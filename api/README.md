# API

## DB

### migrate

- `local`

```bash
$ npm run prisma:local -- migrate dev --name <name>
```

- `devevelopment`

```bash
$ npm run prisma:local -- migrate dev --name <name>
```

- `production`

```bash
$ npm run prisma:prod -- migrate deploy
```

## Reset

- `local`

```bash
$ npm run prisma:local -- migrate reset
```
